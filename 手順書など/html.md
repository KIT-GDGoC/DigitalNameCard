# HTMLの基本 (初心者向け)

HTML（HyperText Markup Language）は、Webサイトの**骨組み**を作るための言語です。
「タグ」と呼ばれる記号を使って、文章や画像に意味を持たせます。

## 1. 基本的な構造

HTMLは、開始タグ `<tag>` と終了タグ `</tag>` で内容を囲むのが基本です。

```html
<p>これは段落です。</p>
```

## 2. よく使うタグ一覧

| タグ | 名称 | 説明 |
| :--- | :--- | :--- |
| `<h1>` 〜 `<h6>` | 見出し | タイトルや見出しに使います。`<h1>`が一番大きく、重要です。 |
| `<p>` | 段落 | 一般的な文章（パラグラフ）に使います。 |
| `<a>` | リンク | 他のページやサイトへ飛ばすときに使います。 |
| `<img>` | 画像 | 画像を表示します。終了タグはありません。 |
| `<div>` | グループ化 | 複数の要素をひとまとめにして、レイアウトを整えるときに使います。 |
| `<ul>`, `<li>` | リスト | 箇条書きを作るときに使います。 |

## 3. 属性（Attribute）

タグに詳細な情報を追加するのが「属性」です。

```html
<!-- href属性でリンク先を指定 -->
<a href="https://google.com">Googleへ</a>

<!-- src属性で画像ファイルを指定 -->
<img src="profile.jpg" alt="プロフィール画像">
```

## 4. ハンズオンでのポイント

今回のハンズオンでは、テンプレートの `index.html` を開いて、以下の部分を探してみましょう。

- `<h1>` の中身を自分の名前に書き換える
- `<p>` の中身を自分の自己紹介に書き換える
- `<a>` のリンク先を自分のSNSアカウントに変える
